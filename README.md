## Features
- List of recipe categories
- Search recipes from the network
- Infinite scroll with pagination
- Save your favorite recipes in a local database
- Share recipes
- Visit original recipe website within a webview


## Android Development
- Jetpack Compose
- Android KTX
- Kotlin Coroutines
- Flow State
- ViewModel
- Navigation
- Room
- Retrofit
- Hilt
- Coil
- DataStore
- Firebase
